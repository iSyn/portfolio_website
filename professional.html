Gamestop Stock Checker  Python, BeautifulSoup, Curses

Gamestop Stock Checker was an app I created when I really wanted to buy a hot new product.
Instead of checking the website every 10 minutes, I researched web python and built an app that checks the site for me. If the item is in stock, it sends the user an email notifying the item is in stock!
